# Mission Control — Releases

Official download hub for **Mission Control**, the desktop AI command center by True Memory Frameworks.

**[⬇ Download the latest release](../../releases/latest)** (macOS, Apple Silicon)

Every build here is code-signed and notarized by Apple.

- Open the DMG, drag **Mission Control** to Applications, launch.
- First run walks you through connecting your builder (any OpenAI / Anthropic / OpenRouter API key works; local models via LM Studio or Ollama too).
- The app self-updates after install.
